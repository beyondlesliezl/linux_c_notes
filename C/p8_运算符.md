# 运算符和表达式

1. 逻辑运算符的短路特性：
    1. 在`||`中，如果前面一个为真，那么就不会计算后面的表达式
    2. 在 `&&`中，如果前面一个为假，那么就不会计算后面的表达式
2. 强制转换

    ```c
    int a;
    float f = 3.9; 
    a = (int)f; 
    pirntf("f = %f\n", f); // 此时f的值还是3.9 
    ```

3. 位运算的重要意义
    1. 将操作数中第n位置1，其他位不变：`num = num | 1 << n`
    2. 将操作数中第n位置0，其他位不变：`num = num & ~(1 << n)`
    3. 测试第n位：`if(num & 1 << n)`
